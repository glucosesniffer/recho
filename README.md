# recho

This is a mini version of the command-line tool echo for Unix systems. Uses clap for parsing command-line arguments.

## Usage

`recho --help`

`recho Hello World`

### Output

```
recho 0.1.0
Ahmad Saeed <ahmadsaeed3290@gmail.com>
Rust echo

USAGE:
    recho [FLAGS] <TEXT>...

FLAGS:
    -h, --help       Prints help information
    -n               Don't print newline
    -V, --version    Prints version information

ARGS:
    <TEXT>...    Input text

```

`Hello World`
